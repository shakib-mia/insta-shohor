JavaScript is an interpreted language. It does not need to be compiled like Java or C++. Browsers' Compiler use Just In Time compilation. It is also called JIT. JIT helps browsers for compiling JavaScript and make executable bytecode that browser can run easily.

Every browsers have two types of storage. One of them is Local Storage and the other is Session Storage.

Local Storage is a permanent solution for storing data. Here data remain saved after the closing browsers.

On the other hand, Session Storage clears data after ending session. It doesn't store data permanently in browser like Local Storage.
